# GET YOUR CARGO.OD

Сайт транспортно-экспедиторской компании **Get Your Cargo.OD**.

## О проекте

Современный адаптивный сайт-визитка для транспортно-экспедиторской компании, предоставляющей услуги по перевозке грузов между Румынией и Украиной.

## Технологии

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (без зависимостей)
- Google Fonts (Inter)

## Структура

```
├── index.html              # Главная страница
├── assets/
│   ├── css/
│   │   └── style.css       # Стили
│   ├── js/
│   │   └── main.js         # Скрипты
│   └── images/
│       └── logo.png        # Логотип
└── README.md
```

## Запуск

Для просмотра сайта достаточно открыть `index.html` в браузере или использовать любой локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```
